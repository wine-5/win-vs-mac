# 戦闘システム設計書

## 戦闘形式
リアルタイムアクション（ハデス・ゼルダ的）

---

## 基本アクション

| アクション | 操作 |
|---|---|
| 移動 | WASD |
| 通常攻撃 | 近接・前方扇形の斬撃 |
| ダッシュ | Shiftキー |
| 必殺技 | 全方位斬り（発動条件・クールタイムは未定） |

---

## 主人公

職業システムは廃止し、主人公は1体固定とする。

| 項目 | 内容 |
|---|---|
| 名前（仮） | セキュリティエージェント |
| 設定 | OSに常駐するセキュリティ機構の擬人化。侵入してきた外部プロセス（Mac軍）を駆除する |
| 攻撃スタイル | 近接・前方扇形（剣士タイプ） |
| 武器 | マウスカーソル（矢印）型の剣 |
| 必殺技 | 全方位斬り |

※ WindowsロゴやDefenderの盾ロゴなどMicrosoftの商標は使用しない。

## ファイル選択システム

ゲーム開始前にエクスプローラーからファイルを最大3つ選択する。
選択したファイルの拡張子に応じてプレイヤーのパラメータが補正される。
補正はInGame開始前に計算・適用完了し、InGame中は変更不可。
レベルアップシステムはなく、ファイル選択がキャラクター強化の唯一の手段。

### 拡張子による補正

| カテゴリ | 対象拡張子 | 補正内容 |
|---|---|---|
| 実行系 | .exe .dll .bat | ATK +10 |
| 文書系 | .txt .pdf .docx | SPD +1.5 |
| 画像系 | .png .jpg .bmp | DEF +8 |
| 音楽系 | .mp3 .wav .flac | HP +20 |
| 圧縮系 | .zip .7z .rar | ATK/SPD/DEF/HP/攻撃射程 小+ |
| 不明系 | それ以外 | 攻撃射程 +20 |

補正値はファイルサイズで変化する（大きいほど補正値が高い・上限あり）。

---

## プレイヤーパラメータ

| パラメータ | 説明 |
|---|---|
| HP | プレイヤーの体力 |
| ATK | 敵に与えるダメージの基礎値 |
| DEF | 敵から受けるダメージの軽減値 |
| SPD | 移動の速さ |
| 攻撃射程 | 攻撃が届く距離 |

※ 防御力は武器には持たせない。DEFはプレイヤーパラメータのみ。

## 敵の設計

### 通常敵

| 敵名 | モチーフ | 行動パターン | 特徴 |
|---|---|---|---|
| Safari | ブラウザ | 遠距離から弾を撃つ | 素早い |
| Xcode | IDE | 動きが遅い・攻撃力高 | 重い |
| Finder | ファイル管理 | 逃げ回る | 回避型 |
| GarageBand | 音楽 | 範囲攻撃（音波） | 範囲型 |
| Siri | アシスタント | 仲間を呼ぶ | 召喚型 |

※ 職業廃止に伴い、攻撃タイプ相性システム（slash/pierce/magic弱点）も廃止。

### 最終ボス
Macをモチーフにしたオリジナルボス。名前は`mac_os.exe`。
Appleの商標・ロゴは使用しない。
行動パターンは未定。
