# ゲームデザイン詳細

## ゲームの流れ
1. Title
2. StageSelect（職業選択 → ファイル選択）
3. Loading（ファイル解析・ダンジョン生成）
4. InGame
5. Result（成功 or 失敗）
6. Titleに戻る

---

## シーン一覧

| シーン名 | UIイメージ | 概要 |
|---|---|---|
| Title | Windowsのログイン画面風 | ユーザーアイコン・ログインボタン風の「START」 |
| StageSelect | エクスプローラー風 | 職業選択ボタン・ファイル選択ボタンの2つを用意。両方の選択完了後にLoadingへ遷移 |
| Loading | Windowsの起動画面風 | プログレスバー＋「ダンジョンを解析中...」などの文言 |
| InGame | サイバー風ダーク空間 | 黒背景・ネオンカラーのダンジョン・3Dアイソメトリック |
| Result | 成功：シャットダウン画面風 / 失敗：エラーダイアログ風 | 成功：「MacBookをシャットダウンしました」/ 失敗：「⚠ GAME OVER - プロセスが停止しました」 |

### StageSelectの遷移フロー
```
StageSelect
├── 職業選択ボタン → 職業選択ダイアログ（剣士・魔法使い・忍者）
└── ファイル選択ボタン → エクスプローラーでファイル・フォルダを選択
↓
両方の選択完了 → Loadingへ遷移
```

## 職業システム

| 職業 | 攻撃スタイル | 必殺技 |
|---|---|---|
| 剣士 | 近接・前方扇形 | 全方位斬り |
| 魔法使い | 遠距離・弾丸 | 前方に巨大魔法弾を1発 |
| 忍者 | 近接・高速 | 分身を出して一斉攻撃 |

---

## ファイル選択システム

ゲーム開始前にエクスプローラーからファイルを最大3つ選択する。
選択したファイルの拡張子に応じてプレイヤーのパラメータが補正される。
補正はInGame開始前に計算・適用完了し、InGame中は変更不可。

### 拡張子による補正

| カテゴリ | 対象拡張子 | 補正内容 |
|---|---|---|
| 実行系 | .exe .dll .bat | ATK +10 |
| 文書系 | .txt .pdf .docx | SPD +1.5 |
| 画像系 | .png .jpg .bmp | DEF +8 |
| 音楽系 | .mp3 .wav .flac | HP +20 |
| 圧縮系 | .zip .7z .rar | ATK/SPD/DEF/HP/攻撃射程 小+ |
| 不明系 | それ以外 | 攻撃射程 +20 |

補正値はファイルサイズで変化する（大きいほど補正値が高い・上限あり）。

## 戦闘システム
参考：ハデス（アイソメトリックアクション）

| 項目 | 内容 |
|---|---|
| 視点 | アイソメトリック（斜め上からの3D俯瞰） |
| 移動 | WASDで自由移動 |
| 通常攻撃 | 職業のロールで攻撃方法が変わる |
| ダッシュ | Shiftキーで短距離ダッシュ |
| 必殺技 | 職業ごとに固定（発動条件・クールタイムは未定） |
| 部屋制 | フォルダ構造で生成された部屋をクリアして進む |
| 敵 | 複数の敵がリアルタイムで動き回る |
| ボス | `mac_os.exe`（Macをモチーフにしたオリジナルボス） |

## 敵の設計

### 敵の生成方針
難易度ごとに敵の種類・パラメータを固定で定義する。

### 通常敵

| 敵名 | モチーフ | 行動パターン | 特徴 |
|---|---|---|---|
| Safari | ブラウザ | 遠距離から弾を撃つ | 素早い |
| Xcode | IDE | 動きが遅い・攻撃力高 | 重い |
| Finder | ファイル管理 | 逃げ回る | 回避型 |
| GarageBand | 音楽 | 範囲攻撃（音波） | 範囲型 |
| Siri | アシスタント | 仲間を呼ぶ | 召喚型 |

### 最終ボス
Macをモチーフにしたオリジナルボス。名前は`mac_os.exe`。
Appleの商標・ロゴは使用しない。

## ダンジョン生成

### 難易度決定ロジック

選択フォルダの合計サイズと階層の深さをスコア化して合算し、3段階の難易度を決定する。

**合計サイズスコア**

| 合計サイズ | スコア |
|---|---|
| 1GB未満 | 0 |
| 1GB〜10GB | 1 |
| 10GB以上 | 2 |

**階層の深さスコア**

| 階層数 | スコア |
|---|---|
| 1〜3階層 | 0 |
| 4〜7階層 | 1 |
| 8階層以上 | 2 |

**合計スコア → 難易度**

| 合計スコア | 難易度 |
|---|---|
| 0〜1 | Easy |
| 2〜3 | Normal |
| 4 | Hard |

### 難易度別ダンジョン構成

| 難易度 | 階層数 | 部屋数 | 敵の強さ |
|---|---|---|---|
| Easy | 1 | 4部屋 | 低 |
| Normal | 3 | 6部屋 | 中 |
| Hard | 5 | 10部屋 | 高 |
