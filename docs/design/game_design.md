# ゲームデザイン詳細

## ゲームの流れ
1. タイトル画面
2. 武器選択画面（武器ベース選択 → 拡張子スロットにファイルを装備）
3. ロード画面（ファイル解析・ダンジョン生成）
4. ゲーム画面
5. リザルト画面（失敗） or クリア画面（ボス討伐）
6. タイトルに戻る

---

## シーン一覧
| シーン名 | 概要 |
|---|---|
| タイトル | ゲーム開始画面 |
| 武器選択 | 武器ベース選択・拡張子スロットへのファイル装備 |
| ロード | ファイル解析・ダンジョン生成 |
| ゲーム | メインのダンジョン攻略画面 |
| ポーズ | ゲーム中の一時停止画面 |
| リザルト（失敗） | ゲームオーバー画面 |
| クリア | ボス討伐後の画面 |

## 武器選択画面
バトル前に武器ベースと拡張子スロットを決定する。一度ダンジョンに入ると変更不可。

```
┌─────────────────────────────────┐
│ DUNGEON DRIVE - WEAPON SETUP    │
├─────────────────────────────────┤
│ 武器ベースを選択:                │
│  ▶ エクスプローラー  [戦士]      │
│    メモ帳           [魔法使い]   │
│    タスクマネージャー[タンク]     │
├─────────────────────────────────┤
│ 拡張スロット（最大3個）          │
│  スロット1: project.exe  ATK+20  │
│  スロット2: music.mp3    HP+15   │
│  スロット3: （空き）              │
├─────────────────────────────────┤
│ 最終パラメータ                   │
│  ATK: 45  DEF: 20  SPD: 30      │
│  HP:  115                        │
└─────────────────────────────────┘
```

## 武器システム

### 武器ベース（Windowsアプリ固定）

| 武器 | モチーフ | ロール | 攻撃方法 | 基礎パラメータ |
|---|---|---|---|---|
| エクスプローラー | フォルダ剣 | 戦士 | 近接・前方扇形 | ATK高・DEF中・SPD低 |
| メモ帳 | 魔法の書 | 魔法使い | 遠距離・弾丸 | ATK中・DEF低・SPD高 |
| タスクマネージャー | 盾 | タンク | 近接・高防御 | ATK低・DEF高・SPD低 |

### 拡張子スロット（バトル前に最大3個装備）
ファイルをスロットに入れることで武器ベースのパラメータを補正する。

| カテゴリ | 対象拡張子 | 補正内容 |
|---|---|---|
| 実行系 | .exe .dll .bat | ATK+ |
| 文書系 | .txt .pdf .docx | SPD+ |
| 画像系 | .png .jpg .bmp | DEF+ |
| 音楽系 | .mp3 .wav .flac | HP+ |
| 圧縮系 | .zip .7z .rar | 全パラメータ小+ |
| 不明系 | それ以外 | 補正なし |

補正値はファイルサイズで変化する（大きいほど補正値が高い・上限あり）。

## 戦闘システム
参考：ハデス（アイソメトリックアクション）

| 項目 | 内容 |
|---|---|
| 視点 | アイソメトリック（斜め上からの3D俯瞰） |
| 移動 | WASDで自由移動 |
| 通常攻撃 | 武器ベースのロールで攻撃方法が変わる |
| ダッシュ | Shiftキーで短距離ダッシュ |
| 部屋制 | フォルダ構造で生成された部屋をクリアして進む |
| 敵 | 複数の敵がリアルタイムで動き回る |
| ボス | `mac_os.exe`（Macをモチーフにしたオリジナルボス） |

## 敵の設計

### 敵の生成方針
難易度ごとに敵の種類・パラメータを固定で定義する。レジストリ参照は行わない。

### 通常敵

| 敵名 | モチーフ | 行動パターン | 特徴 |
|---|---|---|---|
| Safari | ブラウザ | 遠距離から弾を撃つ | 素早い |
| Xcode | IDE | 動きが遅い・攻撃力高 | 重い |
| Finder | ファイル管理 | 逃げ回る | 回避型 |
| GarageBand | 音楽 | 範囲攻撃（音波） | 範囲型 |
| Siri | アシスタント | 仲間を呼ぶ | 召喚型 |

### 最終ボス
Macをモチーフにしたオリジナルボス。名前は`mac_os.exe`。
Appleの商標・ロゴは使用しない。

## ダンジョン生成

### 難易度帯（固定）

| 難易度 | 階層数 | 部屋数 | 敵の強さ |
|---|---|---|---|
| Easy | 1 | 5部屋 | 低 |
| Normal | 3 | 10部屋 | 中 |
| Hard | 5 | 20部屋 | 高 |

### フォルダ解析による難易度決定

| フォルダの情報 | ダンジョンへの反映 |
|---|---|
| フォルダ名 | 部屋の名前 |
| ファイル数・フォルダ数 | 難易度の判定に使用（最終的に3段階に正規化） |
| フォルダの階層の深さ | 難易度の判定に使用（最終的に3段階に正規化） |
