# ゲーム概要（仮）

## コンセプト
「自分のPCが、すべて武器になる」

自分のPCのファイル・フォルダ・システム情報がそのままゲームに反映される
3Dダンジョン攻略アクションゲーム。

---

## ジャンル
3Dトップビュー アクション・ダンジョン攻略

---

## ターゲット
ゲーム好きな一般ユーザー

---

## 世界観
自分のPC内部がデジタル空間として実体化した世界。
プレイヤーはそのデジタル空間に生まれたダンジョンに挑む。

---

## ゲームの流れ（仮）

1. ゲーム開始時にエクスプローラーからファイル・フォルダを選択
2. 選んだファイルの種類・サイズによってプレイヤーのパラメータや武器が決定
3. ファイル構造を元に自動生成されたダンジョンに挑む
4. 起動中のプロセスが敵として出現
5. ボスを討伐してクリア

---

## ファイルと武器・パラメータの対応（検討中）

| ファイル種類 | ゲームへの反映 |
|---|---|
| `.exe` | 攻撃力が高い武器 |
| `.mp4 / .png` | 防御系アイテム |
| `.txt` | 素早さ上昇 |
| ファイルサイズ大 | 重量級・高威力 |
| フォルダの階層の深さ | ダンジョンの階層数に反映 |

---

## Windows API連携（検討中）

| 取得情報 | ゲームへの反映 |
|---|---|
| プロセス列挙 | 敵の種類・数の生成 |
| CPU使用率 | 敵の強さ・行動速度のリアルタイム変化 |
| メモリ使用量 | ダンジョンの環境変化 |
| ファイルシステム | マップ・武器・パラメータの生成 |
