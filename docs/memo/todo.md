### リファクタリング
- System関連のクラスでUpdate内で毎回GetComponentしているため、ISystemに初回取得の純粋仮想関数を追加し、具象クラスでキャッシュする
- 初期化は `{}`、代入は `=` で書き分ける（代入まで `{}` にすると可読性が下がるため）
- DxLib.hを直接インクルードしているが、ラッパーのヘッダーを作って差し替え容易にする
- InGame.cpp のインクルードが多いため専用の集約ヘッダーを作るか検討中
- InGame / InGameView の責務が多く、分離を検討する
- 一度選択した職業やファイルを記憶し、再挑戦時に選択状態を復元する
- [x] シングルトンを全廃し、Applicationをコンポジションルートに（参照注入）
- [x] 敵種ごとの派生クラスを廃止し、データ駆動の汎用Enemyに統一
